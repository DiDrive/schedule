## 准备工作

在开始之前，请确保您：

- ✅ 已登录飞书
- ✅ 有权限在飞书中创建多维表应用

## 创建多维表应用

### 第一步：打开飞书多维表

1. 在飞书左侧导航栏点击 **"云文档"**
2. 点击 **"新建"** → **"多维表格"**
3. 命名为 **"项目排期系统"**
4. 点击 **"创建"**

---

## 创建四个数据表

我们需要创建四个数据表，请按照以下顺序依次创建：

### 📋 表1：人员表（resources）

#### 步骤1：创建表格

1. 在多维表左下角点击 **"+"** 添加新表格
2. 命名为 **"人员"**

#### 步骤2：添加字段

点击表头右侧的 **"+"** 按钮，依次添加以下字段：

| 字段名称 | 字段类型 | 字段ID（必须完全一致） | 必填 | 说明 |
|---------|---------|----------------------|------|------|
| 人员ID | 单行文本 | `id` | ✅ | 系统自动生成的唯一标识 |
| 姓名 | 单行文本 | `name` | ✅ | 人员姓名 |
| 类型 | 单选 | `type` | ✅ | 人员类型（见下方选项） |
| 效率系数 | 数字 | `efficiency` | ✅ | 效率系数（0.5-2.0） |
| 飞书用户ID | 人员 | `feishu_user` | ❌ | 对应的飞书用户 |
| 累计工时 | 数字 | `total_hours` | ❌ | 累计工作小时数 |
| 创建时间 | 日期 | `created_at` | ❌ | 记录创建时间 |
| 更新时间 | 日期 | `updated_at` | ❌ | 记录更新时间 |

#### 步骤3：设置"类型"字段的选项

1. 点击"类型"字段的下拉箭头
2. 点击 **"编辑选项"**
3. 添加以下三个选项：
   - 平面设计
   - 后期制作
   - 物料
4. 点击 **"保存"**

#### 步骤4：设置日期字段格式

对于 `created_at` 和 `updated_at` 字段：
1. 点击字段下拉箭头 → **"编辑字段"**
2. 勾选 **"包含时间"**
3. 点击 **"保存"**

#### 步骤5：隐藏字段

以下字段建议隐藏，避免误修改：
- `id`（人员ID）
- `created_at`（创建时间）
- `updated_at`（更新时间）

**如何隐藏字段**：
1. 点击表头的 **"···"** → **"显示/隐藏列"**
2. 取消勾选上述字段
3. 点击 **"确定"**

## 获取表格信息

创建完四个表格后，需要获取两个重要信息：

### 1. App Token（应用令牌）

**方法一：从 URL 中获取**

1. 在浏览器中打开您的多维表
2. 复制地址栏中的 URL
3. App Token 位于 `https://xxxx.feishu.cn/base/` 和 `/app` 之间

**示例 URL**：
```
https://xxxx.feishu.cn/base/bascnxxxxxxxxxxxx/appxxxxxxxx
```
其中 `bascnxxxxxxxxxxxx` 就是 **App Token**

### 2. Table ID（表格 ID）

**方法一：从表格 URL 中获取**

1. 点击要获取 ID 的表格（如"人员"表）
2. 查看地址栏 URL
3. Table ID 位于 URL 的最后部分

**示例 URL**：
```
https://xxxx.feishu.cn/base/bascnxxxxxxxxxxxx/appxxxxxxxx?table=tblxxxxxxxxxxxx
```
其中 `tblxxxxxxxxxxxx` 就是 **Table ID**

**方法二：从表格设置中获取**

1. 右键点击表格名称（如"人员"）
2. 选择 **"复制表格 ID"**
3. 复制到剪贴板

### 记录您的表格信息

请将以下信息记录下来，配置时需要使用：

| 表格名称 | App Token | Table ID |
|---------|-----------|----------|
| 人员表 | bascnxxxxxxxxxxxx | tblxxxxxxxxxxxx |
| 项目表 | （同上） | tblxxxxxxxxxxxx |
| 任务表 | （同上） | tblxxxxxxxxxxxx |
| 排期表 | （同上） | tblxxxxxxxxxxxx |

**注意**：同一个多维表中的所有表格共享同一个 App Token。

## 配置系统集成

### 第一步：打开飞书集成配置

1. 访问项目排期系统
2. 点击右上角的 **"飞书集成"** 按钮

### 第二步：配置应用信息

切换到 **"应用配置"** 标签页，填写以下信息：

1. **App ID**：从飞书开放平台应用凭证中获取
2. **App Secret**：从飞书开放平台应用凭证中获取
3. **App Token**：填写上面获取的 App Token
4. 点击 **"测试连接"** 按钮
5. 确认显示 **"连接成功"**

### 第三步：配置表格信息

切换到 **"表格配置"** 标签页，填写以下信息：

1. **人员表 Table ID**：填写人员表的 Table ID
2. **项目表 Table ID**：填写项目表的 Table ID
3. **任务表 Table ID**：填写任务表的 Table ID
4. **排期表 Table ID**：填写排期表的 Table ID

### 第四步：启用自动同步（可选）

切换到 **"同步设置"** 标签页：

1. 勾选 **"启用自动同步"**
2. 设置 **同步间隔**（建议 5-15 分钟）
3. 点击 **"立即同步"** 测试同步功能

## 常见问题

### Q1：字段 ID 是什么？怎么设置？

**A**：字段 ID 是系统内部使用的唯一标识符。

**如何查看/设置字段 ID**：
1. 点击字段下拉箭头 → **"编辑字段"**
2. 在字段设置中可以看到 **"字段 ID"**
3. 可以手动修改字段 ID（建议保持默认）

**注意**：字段 ID 设置后不建议修改，否则会影响同步。

### Q2：日期字段为什么要包含时间？

**A**：因为排期系统需要精确到小时级别的时间信息。

**如何设置包含时间**：
1. 点击日期字段的下拉箭头 → **"编辑字段"**
2. 勾选 **"包含时间"**
3. 点击 **"保存"**

### Q3：关联记录字段怎么设置？

**A**：关联记录字段用于建立表格之间的关系。

**示例**：任务表的"项目"字段
1. 点击"项目"字段下拉箭头 → **"编辑字段"**
2. 在"关联表格"下拉框中选择 **"项目"** 表
3. 点击 **"保存"**

这样，每条任务记录就可以关联到一个项目记录。

### Q4：隐藏的字段会影响同步吗？

**A**：不会。隐藏只是前端显示隐藏，数据依然存在，同步功能正常工作。

### Q5：测试连接失败怎么办？

**检查清单**：
- [ ] App ID 和 App Secret 是否正确填写
- [ ] App Token 是否正确填写
- [ ] 飞书应用是否已发布
- [ ] 飞书应用权限是否已批准
- [ ] 网络连接是否正常

### Q6：同步失败怎么办？

**检查清单**：
- [ ] 所有表格的 Table ID 是否正确填写
- [ ] 表格字段 ID 是否与文档一致
- [ ] 是否有必填字段未填写
- [ ] 关联记录字段的配置是否正确
