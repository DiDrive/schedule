## 三种模板的区别

### 模板 1: 字段配置模板 (飞书多维表字段配置模板.xlsx)

**用途**: 用于了解每个表格需要创建哪些字段，以及字段的类型和设置。

**内容**:
- 包含四个工作表：人员表、项目表、任务表、排期表
- 每个字段包含：字段ID、字段名称、字段类型、必填、说明、示例值

**使用场景**:
- 第一次创建飞书多维表时
- 需要了解字段配置细节时

### 模板 2: 示例数据模板 (飞书多维表示例数据模板.xlsx)

**用途**: 包含标准的示例数据，可以直接复制到飞书表格中测试同步功能。

**内容**:
- 包含四个工作表：人员表、项目表、任务表、排期表
- 每个工作表包含标准的示例数据

**使用场景**:
- 需要测试同步功能时
- 需要了解数据格式时
- 需要参考标准数据结构时

### 模板 3: 当前系统数据导出 (项目排期系统数据导出_YYYYMMDD.xlsx)

**用途**: 从您当前系统中导出的人员、项目、任务和排期数据，可直接导入到飞书多维表。

**内容**:
- 包含四个工作表：人员表、项目表、任务表、排期表
- 包含您系统中所有实际数据（自动去重）

**使用场景**:
- 需要将现有数据迁移到飞书时
- 需要备份当前数据时
- 需要与他人共享数据时

## 当前系统数据导出

### 如何导出当前系统数据

**步骤**:
1. 打开项目排期系统
2. 点击右上角的 "配置助手" 按钮
3. 在 Excel 模板生成器区域，点击 **"导出当前系统数据"** 按钮
4. 系统会自动导出当前系统中所有数据

**导出内容包括**:
- **人员表**: 所有人员数据（自动去重）
- **项目表**: 所有项目数据
- **任务表**: 所有任务数据（自动去重）
- **排期表**: 所有排期数据（基于最新的排期结果）

**文件命名**: `项目排期系统数据导出_YYYYMMDD.xlsx`（包含日期）

**数据统计**: 导出后会显示统计信息
```
成功导出系统数据！

人员: X 条
项目: X 条
任务: X 条
排期: X 条
```

### 导出数据的特点

1. **自动去重**: 系统会自动去除重复的人员和任务
2. **完整数据**: 包含所有字段信息，包括状态、优先级、工时等
3. **实时数据**: 导出的是当前系统的最新数据
4. **时间戳**: 文件名包含导出日期，方便区分不同版本

### 使用场景

#### 场景 1: 迁移到飞书多维表

1. 在排期系统中生成排期
2. 点击 "导出当前系统数据"
3. 打开导出的 Excel 文件
4. 将数据复制到飞书多维表中
5. 启用飞书双向同步功能

#### 场景 2: 数据备份

定期导出系统数据，作为备份保存。

#### 场景 3: 数据共享

将导出的 Excel 文件发送给团队成员，便于查看和协作。

#### 场景 4: 数据分析

使用 Excel 的数据分析功能，对项目数据进行分析。

## 如何填写多个数据

### 方法 1: 在 Excel 中直接填写

**步骤**:
1. 打开 `飞书多维表示例数据模板.xlsx`
2. 在现有的示例数据下方添加新行
3. 填写相应的字段数据
4. 保存 Excel 文件

**人员表示例**:

| id | name | type | efficiency |
|----|------|------|------------|
| res-001 | 张三 | 平面设计 | 1.0 |
| res-002 | 李四 | 平面设计 | 1.2 |
| res-003 | 王五 | 后期制作 | 0.9 |
| res-004 | 赵六 | 后期制作 | 1.1 |
| res-005 | 钱七 | 平面设计 | 1.1 |
| res-006 | 孙八 | 后期制作 | 0.8 |

### 方法 2: 在飞书多维表中直接添加

**步骤**:
1. 打开飞书多维表
2. 点击表格底部的 "+" 按钮
3. 直接在表格中输入数据

**注意事项**:
- `id` 字段必须唯一，建议使用格式：`res-001`, `res-002` 等
- `type`、`priority`、`status` 等字段只能从下拉选项中选择，不能手动输入
- `project_id`、`task_id`、`person_id` 等关联字段需要通过关联记录功能选择

## 导入到飞书多维表

### 方式 1: 复制粘贴

**步骤**:
1. 在 Excel 中选中要导入的数据（不包含表头）
2. 按 `Ctrl+C` 复制
3. 在飞书多维表中点击要粘贴的单元格
4. 按 `Ctrl+V` 粘贴

### 方式 2: 使用飞书的导入功能

**步骤**:
1. 在飞书多维表中点击右上角的 "···"
2. 选择 "导入" → "从 Excel 导入"
3. 选择 Excel 文件
4. 按照提示完成导入

**注意**: 关联记录字段（如 `project_id`、`task_id`、`person_id`）可能无法直接导入，需要在导入后手动关联。

## 常见问题

### Q1: 为什么示例数据模板中的关联字段使用的是 ID 而不是名称？

**A**: 因为飞书多维表的关联记录是通过记录ID来关联的，而不是通过名称。这样可以避免同名记录的混淆。

**示例**:
- 正确: `project_id = "proj-001"` (项目ID)
- 错误: `project_id = "XX品牌VI设计"` (项目名称)

### Q2: 如何在飞书中建立关联记录？

**A**:
1. 点击关联字段单元格
2. 在弹出的窗口中选择要关联的记录
3. 点击确定

### Q3: 日期字段为什么要包含时间？

**A**: 因为排期系统需要精确到小时级别的时间信息。

**如何设置包含时间**:
1. 点击日期字段的下拉箭头 → "编辑字段"
2. 勾选 "包含时间"
3. 点击 "保存"

### Q4: 可以删除示例数据模板中的数据吗？

**A**: 可以。示例数据只是参考，您可以根据需要删除或修改。

### Q5: 如何批量添加人员数据？

**A**:
1. 在 Excel 中填写人员数据
2. 复制数据
3. 在飞书人员表中粘贴

**示例**:
```excel
id      name    type        efficiency
res-001  张三    平面设计      1.0
res-002  李四    平面设计      1.2
res-003  王五    后期制作      0.9
res-004  起六    后期制作      1.1
res-005  钱七    平面设计      1.1
res-006  孙八    后期制作      0.8
res-007  周九    物料         1.0
res-008  吴十    平面设计      0.9
```

### Q6: 数据同步失败怎么办？

**检查清单**:
- [ ] 所有表的字段ID是否正确
- [ ] 关联记录是否正确建立
- [ ] 日期字段是否包含时间
- [ ] 单选字段是否使用了正确的选项
- [ ] 必填字段是否都已填写

### Q7: 如何清空测试数据？

**A**:
1. 在飞书多维表中选中所有记录
2. 点击 "删除" 按钮
3. 确认删除

### Q8: 导出的系统数据中为什么没有排期数据？

**A**: 导出的排期数据基于最新的排期结果。如果您还没有生成排期，排期表将是空的。

**解决方法**:
1. 在排期系统中点击 "生成排期" 按钮
2. 等待排期生成完成
3. 再次点击 "导出当前系统数据"

### Q9: 导出数据中的时间格式是什么？

**A**: 导出的时间字段使用 ISO 8601 格式，例如 `2024-01-01T09:30:00.000Z`。

**飞书会自动识别**这种格式为日期时间类型，您无需手动转换。

### Q10: 如何确保导出的数据是最新数据？

**A**: 导出功能会实时从系统中读取最新数据。如果您刚刚添加或修改了数据，建议：

1. 刷新页面
2. 确认数据已保存
3. 点击 "导出当前系统数据"

### Q11: 导出的数据可以修改后再导入到飞书吗？

**A**: 可以。您可以在 Excel 中修改导出的数据，然后导入到飞书。但需要注意：

- **ID 字段**: 不要修改 ID 字段，否则会影响关联关系
- **关联字段**: 确保关联的 ID 存在（如 `project_id` 对应的项目必须存在）
- **选项字段**: 单选字段必须使用预定义的选项值
- **必填字段**: 确保所有必填字段都有值

### Q12: 导出的数据包含所有历史排期吗？

**A**: 不包含。导出的排期数据仅包含**最新的排期结果**，不包含历史排期。

如果您需要保存历史排期，建议：
1. 在每次生成排期后，立即导出数据
2. 文件名中包含日期，便于区分不同版本

## 总结

1. **字段配置模板**: 用于了解如何创建表格字段
2. **示例数据模板**: 包含可直接使用的示例数据
3. **当前系统数据导出**: 从系统中导出实际数据，包含最新的排期结果
4. **填写多个数据**: 直接在现有数据下方添加新行
5. **导入方式**: 复制粘贴或使用飞书导入功能
6. **关联记录**: 使用ID而不是名称
7. **日期字段**: 必须包含时间
8. **数据去重**: 导出时自动去除重复的人员和任务
9. **实时数据**: 导出的是当前系统的最新数据

如果您在配置过程中遇到问题，请查看飞书多维表的创建文档或联系技术支持。
