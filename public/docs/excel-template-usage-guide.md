## 两种模板的区别

### 模板 1: 字段配置模板 (飞书多维表字段配置模板.xlsx)

**用途**: 用于了解每个表格需要创建哪些字段，以及字段的类型和设置。

**内容**:
- 包含四个工作表：人员表、项目表、任务表、排期表
- 每个字段包含：字段ID、字段名称、字段类型、必填、说明、示例值

**使用场景**:
- 第一次创建飞书多维表时
- 需要了解字段配置细节时

### 模板 2: 示例数据模板 (飞书多维表示例数据模板.xlsx)

**用途**: 包含实际的示例数据，可以直接复制到飞书表格中测试同步功能。

**内容**:
- 包含四个工作表：人员表、项目表、任务表、排期表
- 每个工作表包含多条示例数据

**使用场景**:
- 需要测试同步功能时
- 需要了解数据格式时
- 需要批量导入数据时

## 如何填写多个数据

### 方法 1: 在 Excel 中直接填写

**步骤**:
1. 打开 `飞书多维表示例数据模板.xlsx`
2. 在现有的示例数据下方添加新行
3. 填写相应的字段数据
4. 保存 Excel 文件

**人员表示例**:

| id | name | type | efficiency |
|----|------|------|------------|
| res-001 | 张三 | 平面设计 | 1.0 |
| res-002 | 李四 | 平面设计 | 1.2 |
| res-003 | 王五 | 后期制作 | 0.9 |
| res-004 | 赵六 | 后期制作 | 1.1 |
| res-005 | 钱七 | 平面设计 | 1.1 |
| res-006 | 孙八 | 后期制作 | 0.8 |

### 方法 2: 在飞书多维表中直接添加

**步骤**:
1. 打开飞书多维表
2. 点击表格底部的 "+" 按钮
3. 直接在表格中输入数据

**注意事项**:
- `id` 字段必须唯一，建议使用格式：`res-001`, `res-002` 等
- `type`、`priority`、`status` 等字段只能从下拉选项中选择，不能手动输入
- `project_id`、`task_id`、`person_id` 等关联字段需要通过关联记录功能选择

## 导入到飞书多维表

### 方式 1: 复制粘贴

**步骤**:
1. 在 Excel 中选中要导入的数据（不包含表头）
2. 按 `Ctrl+C` 复制
3. 在飞书多维表中点击要粘贴的单元格
4. 按 `Ctrl+V` 粘贴

### 方式 2: 使用飞书的导入功能

**步骤**:
1. 在飞书多维表中点击右上角的 "···"
2. 选择 "导入" → "从 Excel 导入"
3. 选择 Excel 文件
4. 按照提示完成导入

**注意**: 关联记录字段（如 `project_id`、`task_id`、`person_id`）可能无法直接导入，需要在导入后手动关联。

## 常见问题

### Q1: 为什么示例数据模板中的关联字段使用的是 ID 而不是名称？

**A**: 因为飞书多维表的关联记录是通过记录ID来关联的，而不是通过名称。这样可以避免同名记录的混淆。

**示例**:
- 正确: `project_id = "proj-001"` (项目ID)
- 错误: `project_id = "XX品牌VI设计"` (项目名称)

### Q2: 如何在飞书中建立关联记录？

**A**:
1. 点击关联字段单元格
2. 在弹出的窗口中选择要关联的记录
3. 点击确定

### Q3: 日期字段为什么要包含时间？

**A**: 因为排期系统需要精确到小时级别的时间信息。

**如何设置包含时间**:
1. 点击日期字段的下拉箭头 → "编辑字段"
2. 勾选 "包含时间"
3. 点击 "保存"

### Q4: 可以删除示例数据模板中的数据吗？

**A**: 可以。示例数据只是参考，您可以根据需要删除或修改。

### Q5: 如何批量添加人员数据？

**A**:
1. 在 Excel 中填写人员数据
2. 复制数据
3. 在飞书人员表中粘贴

**示例**:
```excel
id      name    type        efficiency
res-001  张三    平面设计      1.0
res-002  李四    平面设计      1.2
res-003  王五    后期制作      0.9
res-004  起六    后期制作      1.1
res-005  钱七    平面设计      1.1
res-006  孙八    后期制作      0.8
res-007  周九    物料         1.0
res-008  吴十    平面设计      0.9
```

### Q6: 数据同步失败怎么办？

**检查清单**:
- [ ] 所有表的字段ID是否正确
- [ ] 关联记录是否正确建立
- [ ] 日期字段是否包含时间
- [ ] 单选字段是否使用了正确的选项
- [ ] 必填字段是否都已填写

### Q7: 如何清空测试数据？

**A**:
1. 在飞书多维表中选中所有记录
2. 点击 "删除" 按钮
3. 确认删除

## 总结

1. **字段配置模板**: 用于了解如何创建表格字段
2. **示例数据模板**: 包含可直接使用的示例数据
3. **填写多个数据**: 直接在现有数据下方添加新行
4. **导入方式**: 复制粘贴或使用飞书导入功能
5. **关联记录**: 使用ID而不是名称
6. **日期字段**: 必须包含时间

如果您在配置过程中遇到问题，请查看飞书多维表的创建文档或联系技术支持。
